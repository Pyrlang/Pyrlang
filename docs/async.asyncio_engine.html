async.asyncio_engine module¶

class pyrlang.async.asyncio_engine.AsyncioEngine¶

Bases: pyrlang.async.base_engine.BaseEngine

Compatibility driver for Asyncio. Create it before creating Node and pass as argument 'engine' like so:

e = AsyncioEngine()
node = Node(name="py@127.0.0.1", cookie="COOKIE", engine=e)
