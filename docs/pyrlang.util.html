util module¶

pyrlang.util.as_str(x: Union[bytes, str, term.atom.Atom])¶

Given byte string, or string, or Atom extracts a string from it. Bytes are interpreted as ASCII/Latin-1.

Raises:Exception -- if argument is not bytes, string or Atom
pyrlang.util.start_pyrlang()¶

This is invoked on import Pyrlang. Function checks OS environment variables documented at Configuring Pyrlang in Runtime.
